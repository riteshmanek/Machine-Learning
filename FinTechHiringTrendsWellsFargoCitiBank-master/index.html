4. Identifying Keywords

  • Data Cleaning :

Before we start calculating keyword scores, the data which has been accumulated needs to be cleaned.

Steps to Clean data :

  • Remove Stop words from the list
  • Perform tokenization and lemmatization
  • Remove Plurals
  • Manually review data

  • Word Count :

As the name suggest, we count the frequency of each word in all the document and extract 100 top words.

  • Text Rank :

Text rank is based on PageRank algorithm. Instead of web pages we use sentences or documents.

Steps to calculate text rank :

  1. Seperate text into sentence
  2. Build matrix of words and count it's appearance in sentence
  3. Construct similarity matrix
  4. Use pagerank algorithm to score sentences.

  • TF-IDF :

Term Frequency (tf): Gives us the frequency of the word in each document in the corpus. It is the ratio of number of times the word appears in a document compared to the total number of words in that document.

It increases as the number of occurrences of that word within the document increases. Each document has its own tf.

TF(t) = (Number of times term t appears in a document) / (Total number of terms in the document).

Inverse Data Frequency (idf): Used to calculate the weight of rare words across all documents in the corpus. The words that occur rarely in the corpus have a high IDF score. It is given by the equation below.

IDF(t) = log_e(Total number of documents / Number of documents with term t in it).

Combining these two we come up with the TF-IDF score (w) for a word in a document in the corpus.

It is the product of tf and idf.
